# Session: 2026-03-25 — Baby in Bloom Build

## What We Built
Complete baby shower web app for Adamary y Juan's party (April 4th).

## Work Completed

### Homepage (page.tsx)
1. **SVG Watercolor Flowers** — Hand-coded all flower components:
   - Cosmos (pink, 3 color palettes), WildRose (larger pink), Daisy (cream + dark center)
   - Sunflower (yellow), SmallBlossom, TinyYellowFlower, Bud
   - Butterfly (3 variants: gold monarch, pink, grey painted lady)
   - Bee (striped body, translucent wings)
   - Leaf, LeafSprig, Stem
2. **Multiple design rounds** improving flower realism:
   - Started with heart-shaped petals → evolved to broad oval ellipses
   - Reduced watercolor blur filters for crisper petal edges
   - Boosted color opacity for richer salmon/pink tones
   - Made butterflies organic with layered wing washes instead of flat fills
   - Added bee near top-right daisy (matching invitation reference)
3. **GSAP Animations** — replaced CSS animations with GSAP for smoothness:
   - Elegant fade-in entrance (not bouncy/poppy)
   - Letter-by-letter "BABY" reveal
   - Falling petal particles with random drift
   - Gold sparkle dots
   - Continuous gentle sway on flower corners
4. **Fonts** — Cormorant Garamond (serif) + Pinyon Script (calligraphy) via next/font
5. **Removed all emojis** from entire app
6. **Updated text** from RSVP style to party context ("celebrating the arrival of")

### Photos Page (photos/page.tsx)
1. **Cookie-based guest identity** — no signup, stored in localStorage
2. **Onboarding flow:** Name → Avatar selection (8 facehash color palettes OR custom photo upload)
3. **Profile filter bar** — horizontal scroll, newest guests first, "All" at end
4. **Polaroid-style photo cards** — uploader avatar in bottom-left white space, no text caption
5. **Instant photo loading** — photo appears in grid immediately after upload
6. **File upload fix** — removed `capture` attribute so both camera and library options show
7. **Realtime subscription** for other users' photos

### Database (Supabase)
- Created `babyshower_guests` table (name, avatar_url, avatar_color)
- Added `guest_id` foreign key to `babyshower_photos`
- Public RLS policies on all tables
